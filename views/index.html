Timestamp Microservice

How to use:

Change the URL path to /api/[date], where date is YYYY-MM-DD, or to Unix timestamp of the input date in milliseconds.

The service will output the date both in UTC and Unix in JSON format!

Example Usage:

  • UTC: /api/2015-12-25
  • Unix: /api/1451001600000
  • Empty date will return invalid: /api/

Example Output:

        
          {"unix":1451001600000, "utc":"Fri, 25 Dec 2015 00:00:00 GMT"}
